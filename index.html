Projects

Vulnerability Manager Plus — Network Device Scanning Module

Role: Software Developer (Jun 2019 – Mar 2023), then Product Manager (Mar 2023 – Present)
Company: Zoho Corporation · Product Page

An enterprise B2B security platform that scans network devices (routers, switches, firewalls) for firmware vulnerabilities, maps them to known CVEs, checks compliance against CIS and NIST benchmarks, and generates remediation reports for IT security teams.

  • Golang core scanning engine: Device discovery, firmware fingerprinting, CVE correlation, and compliance scanning using goroutines for parallel scanning and channels for inter-phase communication.
  • Cgo/JNI interop layer: Bridged Go to the Java-based Zoho platform via Cgo (Go→C) and JNI (C→Java), managing memory safety across three language runtimes.
  • C++ distribution server: Distributes scan tasks across the network, connected to the Golang engine via Cgo.
  • REST API architecture: Versioned endpoints for scan triggers, status checks, configuration, and results — integration surface for ServiceNow and third-party ITSM platforms.
  • Microservices decomposition: Split into four independent services (discovery, fingerprinting, vulnerability correlation, reporting) with independent scaling.
  • PostgreSQL schema: CVE-to-device mappings, scan session histories, compliance baselines, and composite indexes for enterprise query patterns across thousands of concurrent devices.
